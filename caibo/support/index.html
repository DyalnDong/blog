常见问题

提醒推送收不到?

  1. 检查 系统设置 → 财帛 → 通知 是否开启
  2. 检查 我的 → 提醒 是否打开了「开启提醒」总开关
  3. 确认对应卡片填了正确的账单日 / 还款日

首页「近期提醒」怎么用?

首页的「近期提醒」是一个待办清单,分「今日到期 / 未来 7 天到期 / 其他提醒」。点左侧圆圈即可勾掉一条,勾掉后会收进「已完成」分组;想恢复就展开「已完成」点「恢复」。

周转分配方案为空?

推荐会依据每张卡的可用额度计算。到任一卡详情「校准额度」把数字改回银行 App 的真实可用额度即可;或在 我的 → 数据 重置示范卡组一键还原。

AI 功能不工作?

  1. 我的 → AI 设置 → 确认填了 API Key(App 不内置 key,需自带)
  2. 默认 model 是 deepseek/deepseek-chat,可改成 openai/gpt-4o、anthropic/claude-3.5-sonnet 等
  3. 如果报 403,可能是该模型在你当前网络环境受限,换一个 model 即可
  4. 「卡片权益」需要联网搜索,第一次查询较慢属正常;查过的会缓存,下次秒开

怎么添加银行?

账户管理 → 添加银行:内置 15 家主流银行可直接勾选(支持一次多选)。列表里没有的,点「自定义银行」自己填名称、上传 logo 即可。

数据怎么备份?

我的 → 数据 → 导出账户档案 → 生成 JSON → AirDrop 到电脑保存。换机或重装后在同一入口「导入」即可还原(含持卡人、自定义银行、全部卡片)。

支持多设备同步吗?

现版本只支持本机存储 + 手动导出/导入备份。如果你有多设备同步需求,发邮件告诉我们。

已知限制

  • 内置 15 家主流银行;其余可通过「自定义银行」自行添加
  • 仅 iPhone(未做 iPad 优化布局)
  • 需要 iOS 26.0 或更高版本
  • AI 功能需自带 OpenRouter / OpenAI / 自定义兼容服务的 API Key
