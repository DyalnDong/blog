# Twitter(X)KOL 启动方案 · Dylan

这套文档帮你从零启动一个英文 X 账号,目标是攒粉 → 建信任 → 变现(主要赚老外的钱),走轻量、长期、与账号一起成长的路线。

## 核心结论(一眼版)

- **语言**:英文优先(对齐"赚老外钱" + 发挥你的英语护城河 + 顺便练英文)
- **切口**:不会写代码的香港分析师,用 AI(Claude Code)真上架 App + 用同一套系统思维搞钱,**build in public**
- **身份**:半匿名真名(Dylan + 真实战绩,不露脸)
- **变现**:X 拉新 → 导流到自己的 newsletter → 卖产品/订阅/咨询(广告分成只是零头)
- **预期**:6–12 个月慢热,但沉淀的是可复利的长期资产
