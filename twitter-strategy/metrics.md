# Metrics · 每周自查指标 + 里程碑

> 每周日花 10 分钟填一行。看趋势,不看单点。数据决定下周三支柱的配比怎么调。

| W3 | | | | | | | | |

> 重点看三个数:**verified 粉丝**(决定能否开广告分成)、**近 30 天曝光**(冲 3 个月 500 万门槛)、**newsletter 订阅**(你真正拥有的资产)。

## 二、广告分成门槛进度条

- [ ] X Premium 已开
- [ ] 邮箱已验证 + 2FA 已开
- [ ] 资料三件套完整(头像/简介/header)
- [ ] 账号满 3 个月
- [ ] **500 verified 粉丝**(指也付费 X Premium 的粉丝)
- [ ] **近 3 个月累计 500 万曝光**
- [ ] 已绑定 Stripe(收款)
- [ ] 累计收益满 $30(最低提现)

## 三、里程碑

| 时间 | 目标 | 检验 |
|---|---|---|
| 第 1 月 | ~100 粉,找到声音,产出 4 篇 thread | 有 1 条 thread 曝光明显高于平均 = 找到了对的话题 |
| 第 3 月 | 接近广告分成门槛,newsletter 上线 + 50 订阅 | verified 粉爬升、newsletter 有自然增长 |
| 第 6 月 | 1–2K 粉,开广告分成,首笔 affiliate 收入 | 第一笔被动收入到账(哪怕几美元) |
| 第 12 月 | 5K+ 粉,推出首个付费产品/订阅 | newsletter 成为变现主渠道 |

## 四、每月复盘三问

1. **哪个支柱的内容曝光/涨粉最好?** → 下个月加大它的比重(尤其前期可能 build-in-public 远超框架内容)。
2. **哪条 thread 被收藏/转发最多?** → 它值得拆成一个系列,或做成 newsletter 长文 / 未来的付费产品。
3. **我这个月断更过吗?** → 断更=把复利分母按回 1。如果断了,先解决"可持续"再谈"增长"。

## 五、健康度红线(别为了涨粉牺牲这些)

- 不恰烂钱、不推没用过的东西(坦诚是你唯一不可替代的护城河)
- 不为流量编造数字或战绩
- 不把每周时间从轻量(3–5h)无意识滑向重度,导致烧掉热情后断更
