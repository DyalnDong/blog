# Launch Plan · 启动计划 + 运营 SOP + 变现阶梯

> 适配每周 3–5 小时的轻量投入。重点:**少而稳,长期复利**。

## 一、Week 0 搭台清单(一次性,~3 小时)

- [ ] 定 handle(见 `positioning.md`),填好 bio、头像、header(三件套缺一不可,否则不符合广告分成的"完整资料"要求)
- [ ] 开 **X Premium**($8/月,广告分成的前提;也让你自己成为 verified)
- [ ] 验证邮箱 + 开 **2FA**(广告分成硬性要求)
- [ ] 写并**置顶一条"自我介绍 thread"**(草稿见 `starter-kit.md`)
- [ ] 在 **beehiiv 或 Substack** 建一个 newsletter 落地页,链接放进 bio(这是你"拥有的受众",最重要的资产)
- [ ] 关注 **50 个**赛道账号:AI build-in-public、indie hackers、Claude/Cursor 圈、个人理财理性派。建一个 List 方便每天刷
- [ ] 准备排程工具:**Typefully**(免费档够用)或 Hypefury

## 二、每周轻量 SOP(~3.5 小时/周)

**周六批量(~2 小时)** — 全程让 Claude 帮你起草 + 英文润色:
- 写 1 篇深度 **thread**(从博客提炼/翻译,见 `starter-kit.md` 的大纲)
- 起草 **5–7 条短推**(build 进度、一个框架金句、一个失败教训)
- 全部排程到下周(工作日每天发 1 条,thread 选周二或周三早上发)

**每日(15–20 分钟 × 5 天,~1.5 小时/周)**:
- 确认当天的排程推已发出
- **主动回复 5–10 个赛道大号**的推(有观点的真回复,不是"nice!")——这是早期涨粉最快的动作,比发推还重要
- 回复自己推文下的评论

> 节奏总结:**每周 1 thread + 工作日每天 1 短推 + 每天主动互动**。宁可少发也不要断更——断更=把复利分母按回 1。

你已经有 65 篇博客。运营初期**不需要新造内容**,只需把博客"翻译 + 改写成英文推特格式":
- money 类(42 篇)→ 支柱 ②③ 的框架推和失败复盘

## 五、变现阶梯(X 是漏斗顶端,不是终点)

| 阶段 | 粉丝量 | 该做什么 | 预期收入 |
|---|---|---|---|
| **P1 信任期** | 0–500 | 纯免费输出,**不卖任何东西**;开 Premium;目标冲到 500 verified 粉 + 近 3 个月 500 万曝光(广告分成门槛) | $0(投入期) |
| **P2 启动期** | 500–2K | 开**广告分成**;**newsletter 正式运营**(每周一封,把当周最好的 thread + 独家内容发出去);放你**真实在用**的高佣金 affiliate(AI 工具、券商) | 零花钱级 |
| **P3 变现期** | 2K–5K+ | 数字产品 / 付费订阅(如「非程序员用 Claude Code 上架 App」模板包或小课)、付费 newsletter 档、build-with-me 咨询、项目/品牌合作 | $1K–10K/月可期 |

### 变现铁律
1. **X 只负责拉新 → 导流到你拥有的 newsletter → 再卖产品/咨询。** X 算法和账号随时会变,owned audience 才是资产。
2. **P1 阶段绝不卖东西。** 信任没建起来就变现,会把账号做死。
3. **只推你真用过的东西。** 你的核心竞争力是"坦诚",一次恰烂钱毁所有。
4. 高佣金品类正好是你的领域:**软件、金融产品、在线教育**。

## 六、诚实的预期

英文 + 每周 3–5 小时 = **慢热**。通常 **6–12 个月**才到有意义的变现。
但这条路沉淀的是长期、可复利、与你一起成长的资产——这正是你要的"不被流量牵着跑"。
对照你自己写的:复利真正可怕的不是 1%,而是"每天"。**别打断它。**
